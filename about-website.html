About the Website

Code Orange 3476 - teamcodeorange.com

The Code Orange website does not utilize third party themes or drag and drop templates. It was built entirely from ground up by students (and now a graduated mentor) of Code Orange, using purely HTML5/Javascript/CSS.

The website was designed to be responsive, with optimized experiences for phones, tablets as well as traditional browsers. The site is cross-browser compatible, supporting all major browsers including Chrome, Firefox, IE (newer versions) and Safari, as well as their mobile counterparts.

Most of the design work were conducted using Photoshop, and the website itself was coded in Dreamweaver.

The following open source libraries were forked and built upon to provide for a superior experience:

  • jQuery
  • Snap.js for fluid menu drawer support
  • Nicescroll for enhancing scrolling and scrollbars on desktop
  • YT Channel Embed for enhancing youtube video and channel embeds
  • Ninja Slider for enhancing image slider experience
